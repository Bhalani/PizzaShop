#### Configuration
  - `bundle install`
  - `yarn install`

#### Database creation
  - `rails db:create`
  - `rails db:migrate`

#### Database initialization
  - `rails db:seed`

#### How to run the test suite
  - `rspec`

#### How to run the app server and try in browser
  - `bin/dev`

#### In case CSS is not loading properly
  - `rails assets:clobber`
  - `rails assets:precompile`
